Sistema interactivo

Pasos del sistema

Al cargar esta página, se ejecutará automáticamente un script que sigue estos pasos:

  • TASK 1 – Configuración inicial
    • Crear el archivo sistema_interactivo.js
    • Asegurar que se ejecute en el navegador
    • Comentar el código para facilitar su comprensión
  • TASK 2 – Entrada de datos
    • Solicitar el nombre del usuario con prompt()
    • Solicitar la edad con prompt()
    • Guardar los valores en variables usando let o const
  • TASK 3 – Validación
    • Verificar que la edad ingresada sea un número
    • Si no lo es, mostrar un error usando:
    console.error("Error: ingresa una edad válida")
  • TASK 4 – Mensajes condicionales
    • Si la edad es menor a 18, mostrar un mensaje motivador
    • Si la edad es mayor o igual a 18, mostrar un mensaje diferente
    • El mensaje debe incluir el nombre del usuario

Presiona F5 para recargar y ejecutar nuevamente el sistema
